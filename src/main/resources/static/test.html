🚀 InsightOps Dashboard API 테스트

📊 A. 오버뷰 (토글 기능)
Daily/Weekly/Monthly VoC 건수, 증감률, top 카테고리
GET /api/dashboard/overview
🥧 B. Big 카테고리 비중
파이차트용 Big 카테고리별 비중 데이터
GET /api/dashboard/big-category-share
📈 C. 전체 VoC 변화량
라인차트용 시계열 데이터
GET /api/dashboard/total-series
📊 D. Small 카테고리 트렌드
연령/성별 필터링 가능한 막대그래프 데이터
GET /api/dashboard/small-trends
💡 E. 인사이트 카드
AI 기반 주요 인사이트 Top 10
GET /api/dashboard/insights
📋 F. 상담 사례 목록
VoC 상세 리스트 (페이징 지원)
GET /api/dashboard/cases
📧 G. 메일 프리뷰
최근 메일 프리뷰 50개 조회
GET /api/dashboard/mail/recent
✉️ H. 메일 초안 생성
특정 VoC에 대한 메일 초안 생성
POST /api/dashboard/mail/preview

🔧 관리자 기능 (데이터 집계)

voicebot DB 연결 상태 확인
GET /api/admin/voicebot/health
